多行操作
```
Shift + 鼠标中键
```

重命名变量
```
Put text cursor on the variable
Press F2
```

Formating
```
Shift + Alt + F
```

代码折叠
```
折叠所有：Ctrl + K  Ctrl + 0
展开所有：Ctrl + K  Ctrl + J
```

编辑快捷方式
```
Ctrl + K  Ctrl + S
```

打开设置
```
Ctrl + ,
```
